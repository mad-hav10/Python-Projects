# Treasure Island 🏝️💎

Welcome to **Treasure Island**! This is a simple Python adventure game where your goal is to find the treasure by making the right choices.

## 📝 Description

In the **Treasure Island** game, you will make a series of choices to navigate through different paths. 
You will start by deciding whether to go left or right. After that, you'll face more decisions that will 
lead you to either victory or defeat, depending on your choices.

## 🚀 How It Works
The program defines a series of conditional statements to simulate the adventure:
1. You choose whether to go left or right.
2. If you choose to go left, you will be given an option to either swim or wait.
3. If you choose to wait, you'll then be asked to choose a door: red, yellow, or blue.
4. Choosing the correct door (yellow) will result in winning the treasure!

### Sample input and output:
- **Input**: 
  - Option 1: Go left
  - Option 2: Wait
  - Option 3: Choose yellow door
- **Output**: 
  - You win!

## 🖥️ Code Overview
The **Treasure Island** game is implemented in Python. Below is a brief overview of the key parts of the code:

1. **User Decision Handling**: The game starts by asking the player to choose between going left or right.
2. **Conditional Choices**: The player is asked to make further decisions based on previous choices (swim or wait, and which door to choose).
3. **Victory or Defeat**: The game compares the player’s choices to predefined conditions and determines if they win or lose.
