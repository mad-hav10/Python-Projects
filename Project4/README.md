# Rock, Paper, Scissors Game ✊📄✂️

Welcome to the **Rock, Paper, Scissors Game**! This is a Python implementation of the classic hand game where you compete against the computer by choosing rock, paper, or scissors.

## 📝 Description
In the **Rock, Paper, Scissors Game**, you will choose one of the three options: rock, paper, or scissors. The computer also randomly chooses one of the three options, and the game determines the winner based on the following rules:
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

The game will tell you whether you won, lost, or drew based on your choice and the computer's choice.

## 🚀 How It Works
The program simulates the **Rock, Paper, Scissors** game with the following steps:
1. The player is prompted to choose between rock, paper, or scissors.
2. The computer randomly selects one of the options.
3. The game compares the player's choice with the computer's choice and prints the result (win, lose, or draw).

### Sample input and output:
- **Input**: 
  - User chooses: rock, paper, or scissors.
- **Output**: 
  - Computer's choice is printed, and the winner is announced (e.g., "You Win", "Computer Wins", or "Draw").

## 🖥️ Code Overview
The **Rock, Paper, Scissors Game** is implemented in Python. Below is a brief overview of the key parts of the code:

1. **`RPSGame()`**: The main function that handles user input, generates the computer's choice, and determines the winner.
2. **Comparison Logic**: Based on the player's choice and the computer's random choice, the game compares them and prints the result.
3. **Game Flow**: The game handles the logic for each possible outcome: draw, win, or loss.
